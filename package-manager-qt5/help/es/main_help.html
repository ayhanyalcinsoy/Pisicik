Administrador de Paquetes

Administrador de Paquetes se utiliza para buscar, instalar o actualizar paquetes del repositorio de paquetes de Pisi Linux. Es una herramienta práctica y útil para administrar los paquetes de manera sencilla.

Instalando software nuevo

Para instalar un nuevo paquete, seleccione la pestaña "Mostrar paquetes nuevos" y ubique el software que le interesa. Presione "instalar paquetes" y se mostrarán los paquetes y dependencias que se instalarán.

El tiempo de instalación depende de su ancho de banda del acceso a internet. Se mostrará el progreso durante la instalación de los paquetes.

Remover un paquete

Para remover un paquete seleccione la pestaña "mostrar paquetes instalados". Seleccione el paquete a remover y luego presione "Remover paquetes". Se removerán los paquetes seleccionados del sistema.

Mostrar actualizaciones

Por omisión se ve en la barra de tareas las actualizaciones disponibles. Al abrir el administrador de paquetes seleccione "Mostrar actualizaciones" para ver las actualizaciones del sistema disponibles. Presione "Actualizar paquetes" para actualizar el sistema.

Buscar paquete

Para buscar un paquete, simplemente escriba su nombre en la barra de busqueda. Se mostrará una lista de paquetes que coinciden con el término de busqueda.
