Präferenzen

Dieses Fenster wird verwendet, um den Aktualisierungszeitraum und die Depot-Einstellungen zu definieren.

Depots

In diesem Tab können Sie durch verwenden der relevanten Buttons Depots hinzufügen, bearbeiten oder entfernen.

Allgemeine Einstellungen

Hier können Sie die allgemeine Konfiguration für den Pisi Linux Paket-Manager einrichten, wie z.B. das Anzeigen des Ikons im Systemabschnitt in der Taskleiste, Ein-/Ausschalten des Intervalls zur Überprüfung der Paket-Aktualisierungen etc.

Cache

Hier können Sie wählen, wie viel Festplatten-Cache Sie für die heruntergeladene Software verwenden können. Durch das Klicken auf das "Cache leeren" Button wird das ganze heruntergeladene Paket gelöscht und Ihnen mehr Speicherplatz verschaffen. Beachten Sie, dass weder die Systemstabilität beeinflusst, noch die installierte Software gelöscht wird.
